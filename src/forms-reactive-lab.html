Angular 4 Reactive Forms Lab

Instructions

You should already have the Angular 4 solutions repo checked out. Checkout the following branch but DO NOT look at the code!

git checkout forms-reactive

https://github.com/Techolution/TB-Angular4

  1. Start your server if it is not already running and navigate to http://localhost:4200/
  2. Create your own Reactive form that works like this one. It does not have to have the same styling but should have the following functionality:
    • Should display a list of users that can be selected.
    • The user data model should be: id:number, name: string, nickName: string, sex:string, optIn: boolean;
    • Should havae a user detail component that shows the details for the user selected in the user list.
    • The details component should be a reactive form and keep the data model separate from the form model.
    • The user's name in the form should be read only.
    • The nickName should be editable. It is required as well.
    • The nickName field should display styling feedback to the user to let them know the field is required
    • The optIn should be a checkbox. It must be selected as true for the form to be valid.
    • The should be a revert button that will cancel the changes the user has made before clicking update and revert to the data model values.
    • When the form is submitted it should update the data model programmatically.
